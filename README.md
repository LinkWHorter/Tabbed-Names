# Mini-game "Totem Crafting Based on Answers to Questions"

Your task is to select answer options: write them in a message (you can just use numbers), then go to the website and, depending on your selection, select body parts to create a drawing.

The answer will be a photo of the totem (the "Sav" button) and the selected answer options.

Topic: *May Holidays*

> Which May holidays do you usually celebrate?:
>> 1.1. May Day (Labor Day)
>> 1.2. Victory Day
>> 1.3. Easter (if it falls in May)
>> 1.4. Parents' Day
>> 1.5. Just a May weekend to relax

...

> What would you like to do during the next May holidays?:
>> 1.1. Take a short trip
>> 11.2. Visit memorable places
>> 11.3. Spend time with your family in nature
>> 11.4. Organize a theme night
>> 11.5. Try something new and unusual

Now, go to this [website](https://linkwhorter.github.io/Tabbed-Names/) (preferably through a browser, regardless of your device), create a new layer (the "+" button), and choose your options.
Add a new layer for each question.
This way, you should have 11 layers and your final image (example: below).
Layers can be moved, reduced, or enlarged using the "Res" button.
For automatic face positioning (preset "Fac1") and body positioning (preset "Bod2"), use presets with the same names.
